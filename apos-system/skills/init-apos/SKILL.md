# init-apos

1. Detectar raiz real del proyecto.
2. Auditar `.apos/` existente.
3. Crear backup antes de cambios.
4. Crear estructura APOS-X faltante.
5. Clasificar hechos, decisiones, inferencias, riesgos y pendientes.
6. No tocar codigo productivo ni skills globales.
