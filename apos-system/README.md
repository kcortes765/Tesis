# APOS-X

APOS-X is the local-first continuity layer for long projects.

Contract:
- Create memory only when it improves continuity.
- Create skills only when they reduce repeated errors.
- Execute large tasks only under a safe harness.
- Never replace traceability with blind trust in chat.

This folder is the project-local source for APOS-X. It is not installed globally yet.

## Current scope
- Local `.apos/` memory upgrade.
- Codex adapter snippet.
- Skill skeletons.
- Safe-harness seed.
- Migration and eval scaffolding.
