# APOS-X

Este proyecto usa memoria local APOS en `.apos/`.

Para retomar trabajo:
1. Lee `.apos/CONTEXT_POLICY.md`.
2. Lee `.apos/INDEX.md`.
3. Lee `.apos/STATUS.md`.
4. Lee `.apos/HANDOFF.md`.
5. Lee `.apos/PLAN.md`.
6. Lee riesgos y preguntas abiertas si la tarea puede afectar seguridad, ejecucion o decisiones.

Reglas:
- No inventes contexto.
- Distingue hechos, decisiones, inferencias, pendientes y riesgos.
- No pegues chats completos en `.apos/`.
- No modifiques archivos append-only; agrega entradas nuevas.
- Usa `apos-run` para comandos productivos, grandes, destructivos o ambiguos.
- Antes de declarar una tarea terminada, registra evidencia o validacion.
